# F1 2026 — Auto-Updating Calendar

Subscribe in Apple/Google/Outlook Calendar:

```
https://raw.githubusercontent.com/Creative-fw/f1-2026-calendar/main/F1_2026.ics
```

- All 22 Grands Prix, every session — Practice, Sprint Qualifying, Sprint, Qualifying, Race (110 events)
- Times in UTC — your calendar converts to local time automatically
- Races **and** Sprints flagged 🏁 with a 1-hour alert
- Synced daily at 03:45 UTC from the open [sportstimes/f1](https://github.com/sportstimes/f1) dataset (the data behind f1calendar.com), which tracks FIA schedule changes within hours

Part of the same feed family as
[FIA WEC 2026](https://github.com/Creative-fw/wec-2026-calendar),
[ELMS 2026](https://github.com/Creative-fw/elms-2026-calendar) and
[MLMC 2026](https://github.com/Creative-fw/mlmc-2026-calendar).
